# SMART-HEALTH-MONITORING-SYSTEM
IoT-based Smart Health Monitoring System using ESP32 that measures Heart Rate, SpO₂, Body &amp; Room Temperature in real-time, displayed on OLED and monitored remotely via Blynk mobile app.
🏥 Smart Health Monitoring System
An IoT-based health monitoring system built using the ESP32 microcontroller that continuously tracks vital health parameters in real time.
📊 Parameters Monitored

❤️ Heart Rate (BPM) — via MAX30100/MAX30102<br>
🩸 Blood Oxygen Level (SpO₂) — via MAX30100/MAX30102<br>
🌡️ Body Temperature — via DS18B20<br>
🌢️ Room Temperature & Humidity — via DHT11<br>

🔧 Tech Stack

Hardware: ESP32, MAX30102, DS18B20, DHT11, 0.96" OLED<br>
Software: Arduino IDE, Blynk IoT Platform<br>
PCB Design: KiCad<br>
